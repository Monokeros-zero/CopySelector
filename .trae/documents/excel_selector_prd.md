# Excel 指标勾选文件处理工具 - 产品需求文档

## 1. 产品概述

### 1.1 产品背景

在数据分析和业务流程中，经常需要将一个Excel文件中的指标勾选状态映射到另一个Excel文件中。手动处理这种映射工作繁琐且容易出错，需要一个自动化工具来提高效率和准确性。

### 1.2 产品目标

开发一个小型工具，能够：

* 读取源Excel文件中的指标勾选状态

* 根据配置的映射关系，将勾选状态写入目标Excel文件

* 提供直观的GUI界面进行配置和操作

* 确保数据处理的准确性和可靠性

* 支持月份别名配置，处理不同形式的月份表示

* 提供完善的异常处理机制

### 1.3 目标用户

* 数据分析师

* 业务运营人员

* 需要处理Excel文件中指标勾选映射的相关人员

## 2. 功能需求

### 2.1 核心功能

| 功能模块   | 功能描述                               |
| ------ | ---------------------------------- |
| 文件配置   | 配置源文件路径和目标文件路径（支持任意Excel文件） |
| 表头配置   | 配置源文件和目标文件的指标开始行                 |
| 产品映射   | 自动识别源文件中的产品，并提供下拉框选择目标文件的产品映射（对应sheet页） |
| 月份配置   | 提供1-12月勾选，支持月份别名（如"1月"、"1月预测"等） |
| 勾选标记配置 | 配置源文件的勾选标记（如"✓"、"1"等）和目标文件的勾选标记    |
| 配置管理   | 自动保存配置，下次打开工具时加载上次的配置，支持多配置文件管理 |
| 执行映射   | 根据配置执行指标勾选状态的映射，包括清理和写入操作        |
| 结果查看   | 显示处理结果，包括成功映射的指标和错误提示              |
| 打开结果文件 | 一键用默认应用打开结果文件                      |
| 异常处理   | 检测重复产品映射、目标文件无法写入等异常情况          |

### 2.2 详细功能描述

#### 2.2.1 文件配置

* 源文件：支持选择本地Excel文件

* 目标文件：支持选择本地Excel文件，无需固定路径

#### 2.2.2 表头配置

* 源文件产品行：配置源文件中产品名称所在的行号

* 源文件指标开始行：配置源文件中指标开始的行号

* 目标文件指标开始行：配置目标文件中指标开始的行号

#### 2.2.3 产品映射

* 读取源文件中的所有产品并在界面展示

* 每个源产品右侧提供下拉框，选择对应的目标产品

* 下拉框内容为目标文件中的所有sheet页名

* 支持自动匹配同名产品

#### 2.2.4 勾选标记配置

* 源文件勾选标记：支持输入多种标记，如"✓"、"1"、"yes"等

* 目标文件勾选标记：可配置要写入的勾选标记

### 2.2.5 月份配置

* 月份选择：提供1-12月的勾选框，用户可以选择需要处理的月份

* 月份别名支持：通过month.json配置文件支持月份的不同表达方式（如"1月"、"1月预测"、"1月预估"等）

* 月份列识别：自动识别目标文件中对应月份的列位置，支持月份出现在第1行或第2行的情况

#### 2.2.6 执行映射

* 点击执行按钮后，系统根据配置处理数据

* 先清空目标文件中对应产品sheet的指标行的所有月份列标记

* 源文件产品下勾选的指标，在目标文件的映射产品sheet页中找到对应指标行

* 在勾选的月份列中打上配置的勾选标记

* 没有目标产品映射的源产品不处理

* 目标产品没有对应指标时，在界面打印提示信息，但不打断处理流程

#### 2.2.7 结果查看

* 执行成功后弹出提示框，显示处理完成信息

* 提供选项，一键打开结果文件查看

* 控制台显示详细的处理信息和错误提示

#### 2.2.8 配置管理

* 自动保存用户的所有配置，包括文件路径、表头配置、产品映射、月份选择和勾选标记
* 下次打开工具时自动加载上次的配置，无需重新设置
* 支持保存多个配置文件，通过下拉框选择加载
* 配置文件存储在configs目录下，确保数据安全

#### 2.2.9 异常处理

* 检测重复的产品映射并弹出错误提示
* 处理目标文件无法写入的情况（如文件被打开）并弹出错误提示
* 处理目标文件中不存在的指标，打印提示信息但不打断处理流程

## 3. 非功能需求

### 3.1 性能要求

* 处理速度：对于1000行以内的Excel文件，处理时间不超过5秒

* 内存占用：运行时内存占用不超过200MB

### 3.2 可靠性

* 数据处理准确性：确保勾选状态正确映射

* 错误处理：对无效输入和异常情况有合理的错误提示

* 数据安全：配置文件存储在本地，确保数据安全

### 3.3 易用性

* 界面简洁直观，操作流程清晰

* 提供配置示例和帮助信息

* 支持常见Excel格式（.xlsx）

### 3.4 兼容性

* 支持Windows、macOS等主流操作系统

* 支持Excel 2016及以上版本生成的文件

## 4. 数据结构

### 4.1 源文件结构

* 包含产品和指标的二维表格

* 指标在列上，从指定行开始

* 产品在指定行，从第二列开始

* 勾选状态通过特定标记表示

### 4.2 目标文件结构

* 每个产品对应一个sheet页

* 指标在列上，从指定行开始

* 月份在第1行或第2行，可能包含别名（如"1月预测"）

* 每个指标行对应需要标记的月份列

### 4.3 配置数据结构

```javascript
{
  "source_file": "path/to/source.xlsx",
  "target_file": "path/to/target.xlsx",
  "source_config": {
    "category_row": 1,              // 产品行号
    "indicator_start_row": 3        // 指标开始行号
  },
  "target_config": {
    "indicator_start_row": 3        // 指标开始行号
  },
  "check_marks": {
    "source": ["✓"],  // 源文件勾选标记
    "target": "✓"     // 目标文件勾选标记
  },
  "category_mapping": {           // 产品映射关系
    "产品B1": "产品B1",
    "产品B2": "产品B2",
    "产品B3": "产品B3"
  },
  "month_config": {
    "selected_months": [3, 4, 5, 6, 7, 8, 9, 10]  // 选择的月份
  }
}
```

## 5. 界面设计

### 5.1 布局结构

* 顶部：配置管理区域（配置下拉框、加载配置、保存配置）

* 中部：
  * 左侧：文件配置、产品行配置、指标配置、标记配置
  * 右侧：产品映射区域（源产品列表和目标产品下拉选择）
  * 底部：月份选择区域（1-12月的勾选框）

* 底部：操作按钮区域（执行、查看结果）

### 5.2 交互流程

1. 启动工具，自动加载上次配置
2. 选择源文件和目标文件
3. 设置参考文件产品行
4. 点击"加载产品"按钮加载产品信息
5. 为每个源产品选择对应的目标产品（sheet页名）
6. 配置源文件和目标文件的指标开始行
7. 配置源文件和目标文件的标记
8. 选择需要处理的月份（1-12月）
9. 点击"执行"按钮开始处理
10. 处理完成后，弹出提示框，可选择打开结果文件查看

### 5.3 视觉设计

* 配色方案：简洁的浅色主题，主要使用系统默认颜色

* 字体：无衬线字体，清晰易读

* 按钮样式：标准系统按钮样式

* 布局：合理利用空间，左右分布主要配置区域

## 6. 操作流程

### 6.1 配置流程

1. 启动工具（自动加载上次配置）
2. 点击"浏览"按钮选择源文件
3. 点击"浏览"按钮选择目标文件
4. 在"产品行配置"区域，输入参考文件产品行号
5. 点击"加载产品"按钮，系统读取源文件和目标文件的产品信息
6. 在产品映射区域，为每个源产品从下拉框中选择对应的目标产品
7. 在"指标配置"区域，输入参考文件和目标文件的指标开始行号
8. 在"标记配置"区域，输入源文件和目标文件的标记
9. 在"月份选择"区域，勾选需要处理的月份

### 6.2 执行流程

1. 点击"执行"按钮
2. 系统检查是否有重复的产品映射，如有则弹出错误提示
3. 系统读取源文件中的勾选状态
4. 系统读取目标文件的sheet页名作为产品
5. 系统识别目标文件中对应月份的列位置（支持第1行和第2行）
6. 系统清空目标文件中对应产品sheet的指标行的所有月份列标记
7. 根据产品映射关系，在目标文件的对应sheet页中找到指标行
8. 在勾选的月份列中打上配置的勾选标记
9. 显示处理结果，包括成功映射的指标数量和错误信息
10. 弹出提示框，可选择打开结果文件查看

* 能够支持月份别名配置

### 7.2 性能验收

* 处理1000行以内的Excel文件，响应时间不超过5秒

## 8. 风险和约束

### 8.1 风险

* Excel文件格式复杂，可能存在解析错误

* 大型Excel文件可能导致性能问题

* 用户配置错误可能导致映射结果不正确

* 目标文件被其他程序打开可能导致写入失败

### 8.2 约束

* 仅支持.xlsx格式的Excel文件

* 不支持复杂的Excel公式和宏

* 源文件和目标文件的结构需要符合预期格式

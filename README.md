# Excel 指标勾选文件处理工具

## 项目介绍

Excel 指标勾选文件处理工具是一个用于将参考Excel文件中的指标勾选状态映射到目标Excel文件的自动化工具，提供了直观的GUI界面和完善的异常处理机制。

## 功能特性

- **读取源Excel文件**中的指标勾选状态
- **根据配置的映射关系**，将勾选状态写入目标Excel文件
- **支持月份维度**的勾选配置
- **月份别名支持**：通过month.json配置文件支持月份的不同表达方式（如"1月"、"1月预测"等）
- **自动保存和加载配置**，下次打开时记住上次的设置
- **GUI界面**：提供直观的用户界面，支持文件选择、产品映射配置等
- **异常处理**：
  - 检测重复的产品映射并提示
  - 处理目标文件无法写入的情况（如文件被打开）
  - 打印目标文件中不存在的指标，不打断处理流程
- **灵活的月份处理**：支持目标文件中月份出现在第1行或第2行的情况
